# Elena & Riccardo — Wedding Site

A single-page, bilingual (IT/EN) wedding website. Static — no build step.

## Files
- `index.html` — the whole site
- `assets/` — all photos it uses

## Put it on GitHub + publish free with GitHub Pages
1. Create a new repository on GitHub (e.g. `wedding`).
2. Upload **`index.html`** and the **`assets/`** folder (keep the folder structure).
3. In the repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, pick `main` / `/root`, Save.
4. Wait ~1 minute — your site is live at `https://<username>.github.io/wedding/`.

The page auto-detects the browser language (Italian → IT, everything else → EN) and has an IT · EN switch in the top bar.
